Список користувачів

Файл:

/users/users.json

У файлі зберігається об'єкт:

{
  "user_new_index": 1,
  "users": []
}

Властивість user_new_index вказує на індекс останнього користувача +1.

Властивість users - масив користувачів:

{
  "id": "u0",
  "name": "boba",
}
Параметр Опис
id Кодова назва користувача, назва папки з файлами
name Ім'я (логін) користувача
