Повернутися до Вікі > JSON-файли проекту

Світи користувача

Файл:

/users/%username%/worlds.json

У файлі зберігається об'єкт:

{
  "id": "u0",
  "name": "boba",
  "world_new_index": 1,
  "worlds": []
}

Властивість id - код користувача, назва його папки.

Властивість name - логін користувача.

Властивість world_new_index вказує на індекс останнього створенного світу +1.

Властивість worlds - масив збережених світів. Кожен світ - об'єкт такої структури:

{
  "id": "w0",
  "version": "0.0.0",
  "name": "Мій перший світ",
  "date_created": "2019-01-04 19:43:37",
  "date_saved": "2019-01-07 14:17:22",
  "time": "7568",
  "dwarves": "4/3/2"
}
Параметр Опис
index Кодова назва світу, назва папки з файлами
version Версія світу (для конвертації в новіші)
name Ім'я світу, що задає користувач
date_created Дата створення світу
date_saved Дата останнього збереження світу
time Загальний ігровий час (в ігрових одиницях часу - тіках, =6 хв.)
dwarves Кількість дворфів: дорослих / дітей / мертвих
